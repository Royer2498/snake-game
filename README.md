# Snake (JavaScript + HTML)

A minimal Snake game implemented with plain HTML, CSS and JavaScript.

## Files
- `index.html` — main page / canvas
- `index.css` — styles
- `index.js` — game logic

## How to run
1. Open `index.html` in a modern browser (Chrome, Edge, Firefox).
2. Or serve the folder with a simple HTTP server:
   - PowerShell: `python -m http.server 8000` (then open http://localhost:8000)

## Controls
- Arrow keys or WASD to move
- R key or "Restart" button to restart

## Notes
- Grid size: 20px tiles, canvas 400×400.
- Game ends on wall or self-collision; score shown top-left.
